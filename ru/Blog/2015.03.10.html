Год второй, месяц шестой, день двадцать пятый.

Я записал песн... кхм.. музык.... ээмм... в общем трек. Наконецт опробовал конфигурацию, с помощью которой я собираюсь мутить аккомпанемент для концертов с #DRSSTC. Для записи использовался из инструментов только голос, из оборудования только микрофон, из софта: бидуля, мёбиус и торнадо :3 Трек полностью импровизированный, все эффекты накладывались в реальном времени. Из компьютерной обработки только рекорд, стоп и широкополосный компрессор с лимитером когда делал мастеринг уже готового трека.
